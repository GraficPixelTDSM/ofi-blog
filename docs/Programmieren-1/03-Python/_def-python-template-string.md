:::def formatierter Text `f''`
Variablenwerte lassen sich mit geschweiften Klammern `{}` in einen Text einfügen. Dazu muss zu Beginn des Textes ein kleines `f` (für "fromatierten Text") hinzugefügt werden.

```py live_py slim
name = 'Alfred'
print(f'Hallo {name}')
```
:::
