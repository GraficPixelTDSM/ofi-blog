# Zeichencodierung

Datenpakete bestehen aus verschiedenen Zahlen und Zeichen. Wie lassen sich diese in Einsen und Nullen übersetzen? Auf dem Computer ist jedem Zeichen eine eindeutige Zahl zugeordnet - eine sog. **Zeichencodierung**. Eine gängige Zeichencodierung ist etwa [ASCII, 128 Zeichen](https://de.wikipedia.org/wiki/American_Standard_Code_for_Information_Interchange), wobei für ein Zeichen genau `7 bit`, also 7 Stellen mit `0` oder `1`, notwendig sind. In diesen 128 möglichen Zeichen sind `33` Steuerzeichen und `95` normale Zeichen enthalten.

**Steuerzeichen** sind spezielle Zeichen wie bspw. das Zeichen `\n` für einen Zeilenumbruch oder `\t` für einen Tabulator.

Die ASCII-Codierung enthält noch keine Umlaute, weshalb mit [Latin 1, 256 Zeichen](https://de.wikipedia.org/wiki/ISO_8859-1) der Informationsgehalt um ein Bit erhöht wurde, so dass insgesamt `256` Zeichen codiert werden können - die ersten `128` Zeichen sind dabei identisch mit ASCII.

## Python

Auch in Python werden im Hintergrund Zahlen für Buchstaben und Binärcodes für Dezimalzahlen verwendet.  kann in Python einfach in Erfahrung gebracht werden mit der Funktion `ord`.

:::def `ord(<Buchstabe>)`
Die sogenannte **Ordnungszahl** für einen (einzelnen!) Buchstaben kann mit dem Befehl `ord` bestimmt werden.
```py live_py slim
zahl = ord('A')
print(zahl)
```
:::

:::def `bin(<Zahl>)`
Eine Dezimalzahl kann in eine Binär-Zahl umgewandelt werden.
```py live_py slim
binaer = bin(65)
print(binaer)
```

:::finding Hinweis
Mit dem `0b` zu Beginn zeigt Python an, dass es sich um eine Binärzahl handelt. Analog zu der [Subskript-Schreibweise](03-Zeichencodierung.mdx#hinweis-subskript) $1000001_2$ schreibt Python `0b1000001`.
:::
:::

:::def `int(<Binärzahl>)`
Mit dem Befehl `int` (abkürzung für *Integer*, also *Ganze Zahl*) kann eine Binärzahl in eine Dezimalzahl übersetzt werden:
```py live_py slim
zahl = int(0b1001)
print(zahl)
```

Wenn das `0b` bei der Binärzahl nicht geschrieben wird, sieht die Syntax wie folgt aus:

<h5 className="admonition-heading">

`int(<Zahl>, 2)`

</h5>

Mit dem Befehl `int` (abkürzung für *Integer*, also *Ganze Zahl*) kann eine Binärzahl in eine Dezimalzahl übersetzt werden:

```py live_py slim
zahl = int(1001, 2)
print(zahl)
```
:::finding Hinweis
Der zweite Parameter gibt die Basis des Zahlensystems an, hier also für Binär `2`.
:::

:::

:::def `chr(<Zahl>)`
Gibt den zu einer Zahl gehörenden Buchstaben zurück.
```py live_py slim
buchstabe = chr(77)
print(buchstabe)
```
:::
